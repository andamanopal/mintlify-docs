# Alto CERO API Reference

Welcome to the Alto CERO API documentation. This reference provides details about all available API endpoints, request/response formats, and includes interactive examples that let you test the API directly from the documentation.

## Base URL

All API endpoints use the following base URL pattern:

```
https://api.altotech.ai/api/
```

For local development environments:

```
http://localhost:8000/api/
```

## Response Format

All API responses follow a consistent JSON format:

```json
{
  // For successful operations
  "data": {
    // Response data object or array
  },
  "message": "Operation successful",
  
  // For paginated responses
  "count": 10,        // Number of items in the current response
  "total": 45,        // Total number of items available
  "page": 1,          // Current page number
  "page_size": 10,    // Number of items per page
  
  // For error responses
  "error": {
    "code": "ERROR_CODE",
    "message": "Error description",
    "details": {}     // Optional error details
  }
}
```

## Rate Limiting

The API implements rate limiting to ensure service stability. Limits vary by endpoint:

- Authentication endpoints: 10 requests per minute
- Read operations: 60 requests per minute 
- Write operations: 30 requests per minute

Rate limit information is included in the response headers:

```
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 45
X-RateLimit-Reset: 1616799429
```
